### Installation
1.  Clone the repo
    ```
    git clone https://github.com/ILooL21/Detective-Den-Conan-Merch-Shop.git
    cd Detective-Den-Conan-Merch-Shop
    ```
2. Buka 2 terminal
   
   Client
   ```
   cd client
   npm install
   npm run dev
   ```
   Api
   ```
   cd api
   npm install
   npm run dev
   ```
